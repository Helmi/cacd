---

## Requirements

- Node.js 18 or later
- One or more AI coding assistants installed (Claude Code, Gemini CLI, etc.)

## Install via npm

```bash
npm install -g cacd
```

## Run without installing

```bash
npx cacd
```

## Verify installation

```bash
cacd --version
```
